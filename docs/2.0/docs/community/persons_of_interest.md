## 责任 [¶](#responsibility "Permalink to this header")



* 分类并修复分配给模块或库的高优先级问题
* 分类、审查和落实分配给模块或库的高优先级拉取请求
* 在 [discuss.pytarch.org](https://discuss) 和 [dev-discuss.pytorch.org](https://dev-discuss.pytorch.org/) 上回答模块或库问题
* 维护公共用户和开发文档
* 每半年或每季度召开会议并分享会议记录和路线图基础
